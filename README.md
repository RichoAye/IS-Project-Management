#  Smart Healthcare Management System

## Project Overview
This project is a Smart Healthcare Management System designed to improve hospital services. It enables real-time patient monitoring and medical equipment tracking using modern information system concepts.


## Objectives
- Monitor patient health in real-time  
- Track hospital medical equipment  
- Improve hospital efficiency  
- Support healthcare staff decision-making  


##  Tools Used
- Trello (Project Management)  
- GitHub (Version Control & File Management)

##  Project Management Process
- Tasks were organized using Trello boards  
- Work was divided into To Do, In Progress, Testing, and Completed  
- Team members were assigned specific responsibilities  
- Progress was tracked using comments and task movement
